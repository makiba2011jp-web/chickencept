# カード画像の置き場所

このフォルダに、下記の**正確なファイル名**で画像を保存してください。
**PNG・JPG どちらでもOK**（`.png` を先に探し、無ければ `.jpg` を自動で読み込みます）。
無いカードは画像なしのテキスト表示になります。

| サラマンダー   | 火 | `salamander.png` / `.jpg` |

## メモ
- PNG・JPG どちらでも可（拡張子はそのままでOK、変換不要）
- できれば正方形（例 512×512）にトリミングすると、手札カードがきれいに収まります
- 1枚あたり数百KB〜2MB程度を目安に（大きすぎると読み込みが重くなります）
- ファイルを追加したら commit & push で公開サイトに反映されます

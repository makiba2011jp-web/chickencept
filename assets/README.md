# 音楽・効果音の置き場所

## BGM（2曲）
画面に応じて自動でループ再生・切り替わります（最初のクリック後に開始）。

| ファイル名 | 流れる場面 |
|---|---|
| **`assets/title.mp3`** | タイトル画面・待機ロビー |
| **`assets/bgm.mp3`** | 対戦中（ゲーム開始後） |

- ファイル名は必ず小文字で上記の通り
- どちらか／両方が無くても無音のまま動きます（エラーにはなりません）
- 画面右上の 🔊 ボタンでオン/オフ切替（設定は記憶されます）

## 無料で使えるBGMの入手先（日本語・商用可のものが多い）
- DOVA-SYNDROME … https://dova-s.jp/
- 甘茶の音楽工房 … https://amachamusic.chagasi.com/
- 魔王魂 … https://maou.audio/

ダウンロードしたmp3を `bgm.mp3` にリネームしてここに置くだけです。
（利用規約・クレジット表記の条件は各サイトで確認してください）

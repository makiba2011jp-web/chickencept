# プレイヤーの駒画像

このフォルダに、参加順（席順）で駒画像を置きます。
**PNG・JPG どちらでもOK**（`.png` を先に探し、無ければ `.jpg`）。
画像が無い席は、これまで通り色付きの丸で表示されます。

| 席（参加順） | ファイル名 | 既定の色 |
|---|---|---|
| 1人目（ホスト） | `p1.png` / `p1.jpg` | 青 |
| 2人目 | `p2.png` / `p2.jpg` | 赤 |
| 3人目 | `p3.png` / `p3.jpg` | 緑 |
| 4人目 | `p4.png` / `p4.jpg` | 橙 |

## メモ
- 正方形（例 256×256）が一番きれいに収まります（丸くトリミング表示されます）
- 盤面の駒・上部のプレイヤー欄の両方に表示されます
- 追加したら commit & push で公開サイトに反映されます
